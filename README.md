主要技术栈：
 - vuejs2.0
 - vue-router
 - vuex
 - vue-resource
 - mint-ui

由于本人的服务器有点渣，后台通过代理返回酷狗列表页这个阶段所需时间极不稳定，所以部分页面采用本地json模拟数据<br>
由于是单页应用，在路由切换时不会暂停音乐播放，是一个小亮点，也可以实现播放列表等功能，可能会加上。
代码暂未优化，耦合有点大，会优化的。<br>
本demo会长期更新，如果觉得还可以的话，给颗star吧。^9^

如何使用
----

 1. 下载本项目到本地
 2. npm install下载依赖
 3. npm run dev
